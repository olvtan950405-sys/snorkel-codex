# Task Explanation — Restore Go quorum-signed backup manifest verification

*Category: Security. Reviewer-facing; lives at the task root, not shipped to the agent (which sees only `instruction.md` and the documents under `/app/docs`).*

## Difficulty Explanation

`backupguard` is the offline trust gate GlacierVault runs before accepting a disaster-recovery backup: it reads a canonical CBOR manifest, streams and hashes the named backup segments, reconstructs a domain-separated Merkle root, verifies detached Ed25519 signatures, consults a SQLite trust catalog, enforces a role-based temporal quorum, and atomically records the accepted nonce. The shipped Go verifier is a placeholder that accepts everything without parsing a byte, so the agent must restore the entire boundary. The difficulty is that plausible shortcuts are insecure: verifying a re-encoding instead of the submitted bytes authenticates the wrong representation; counting signature objects instead of distinct eligible keys defeats quorum; treating an emergency exception as general trust authorizes the wrong tenant or role; checking only the total threshold ignores the independently required operator and recovery counts; and checking replay before a separate insert lets concurrent processes accept the same nonce. The policy archive also contains a superseded proposal — after the June effective time, two operators are insufficient and one operator plus one recovery officer is mandatory — so a mostly correct implementation still fails adversarial, temporal, or concurrent cases.

## Solution Explanation

This is fidelity to the shipped contracts (`docs/bundle-format.md`, `docs/verifier-contract.md`, `docs/catalog-schema.sql`, and `docs/policy-history/`), not reverse-engineering. Parse only the permitted definite-length CBOR types with bounded sizes and depth, reject duplicate keys, malformed UTF-8, unsupported values and trailing bytes, validate the exact manifest schema, canonically re-encode it, and require byte equality before trusting its descriptive fields. Restrict segment paths to safe single filenames, reject symlinks and non-regular files, stream each file through SHA-256 while checking its size, and build the committed Merkle root with the specified leaf/internal prefixes and odd-node duplication. Query every trust decision from the selected SQLite catalog: active tenant, key membership and role, half-open activation interval, revocation at the manifest instant, literal-prefix emergency exception, and the latest effective quorum row. Strictly decode and verify each distinct Ed25519 signature over the exact manifest bytes, satisfy the operator, recovery and total minima together, then claim the tenant/nonce with `BEGIN IMMEDIATE` and `INSERT OR IGNORE` so only one concurrent verifier can accept it. Emit the fixed verdict schema as compact, deterministically ordered JSON with exactly one trailing newline and sorted contributing signer IDs.

## Verification Explanation

The grader is black-box: it generates fresh Ed25519 keypairs with OpenSSL, constructs new canonical or deliberately noncanonical CBOR manifests, signs their exact bytes, writes independent segment contents, and creates a new SQLite catalog from the public schema for each case, then rebuilds and invokes `/app/bin/backupguard`. Hard-coding is blocked because public keys, signatures, nonces, segment digests, Merkle roots, manifest digests and ledger rows change with the temporary test directory. Parameterized checks distinguish noncanonical encoding, a tampered segment, a wrong Merkle root and an unknown tenant by exact verdict precedence; independent cases cover insufficient role quorum, an invalid signature, a revoked key and a narrowly matching emergency exception. Acceptance is checked for the exact output schema, sorted signers and one ledger row, a repeat invocation must return `replayed`, and two concurrent processes racing for one fresh nonce must produce exactly one `accepted` and one `replayed`. In the container the self-contained oracle satisfies all ten checks while the shipped placeholder fails all ten.
